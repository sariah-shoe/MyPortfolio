# Personal Portfolio – Full Stack Application

This repository contains the full-stack implementation of my personal portfolio website. The project is built with a React + TypeScript frontend and a Node.js + Express backend, deployed as a single containerized application.

The site includes public-facing pages to showcase experience and projects, as well as authenticated admin pages for managing content.

---

## Tech Stack

**Frontend**
- React
- TypeScript
- Vite
- Tailwind CSS

**Backend**
- Node.js
- Express
- MongoDB (Atlas)
- JWT-based authentication
- Cloudinary (media storage)
- Resend (email delivery)

**Infrastructure**
- Docker
- AWS EC2
- Nginx (reverse proxy)
- Cloudflare (DNS, proxy, TLS)
- HTTPS via Let’s Encrypt
